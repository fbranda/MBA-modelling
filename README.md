# MBA-modelling
This repository includes simulated datasets and scripts required to reproduce all of the analyses for the article entitled:
Modelling serological multiplex bead assays responses: A case study from Malaysia
The repository structure is organized as follows:
MBA modelling/
│
├── Input data/
│   ├── simulate_MBA_data.R
│   ├── data_MBA_modelling.csv
│   └── data_MBA_modelling.txt
│
└── R script/
    ├── code_MBA_modelling.R
    ├── crp_gibbs_fun.R
    ├── mesh_spaziale.rds
    └── gadm41_MYS_shp/
        └── [Malaysia shapefiles]
Specifically, they are:

i) The folder “Input data” stores the R simulation script (simulate_MBA_data.R), the input dataset generated by it in comma delimited format (data_MBA_modelling.csv), and its associated codebook (data_MBA_modelling.txt).

ii) The folder “R script” stores the main analysis script (code_MBA_modelling.R) required to run the analysis on the simulated data, alongside crp_gibbs_fun.R, which contains the functions needed to run the Dirichlet Process Mixture Model (DPMM) within the main analysis, and the spatial mesh object (mesh_spaziale.rds) needed to define the spatial domain for the analysis. Additionally, it contains the subfolder “gadm41_MYS_shp”, which stores the shapefile of Malaysia downloaded from the GADM spatial database (https://gadm.org/download_country.html).

Please note that because the analyses are run on simulated data created to mirror the structure of the original data, the resulting numerical values and plots will not be identical to those presented in the published paper.
